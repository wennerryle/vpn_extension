How to Set Up Secure DNS in Microsoft Edge

  1. Click the or enter edge://settings/privacy into your browser's address bar.

    You can also open the above page manually: Menu → Settings → Privacy, search and services.

  2. Scroll down to the Security section.

    Enable the Use secure DNS to specify how to lookup the network address for websites.

    Switch to Choose a service provider.

    Click an empty box and select Google (Public DNS) or Cloudflare (1.1.1.1).

  3. Be sure to restart your browser.
